KONTROLE

W / A / S / D — Premikanje (Naprej / Levo / Nazaj / Desno)
SPACE (Preslednica) — Plavaj GOR
SHIFT / CTRL / C — Plavaj DOL
MIŠKA (Drag) — Obračanje pogleda (Swipe)
DESNI KLIK — Zgodba ribice (Start / Stop)
ESC — Pavza / Meni
